# Fitts' Law Task

フィッツの法則を体験するブラウザ実験課題。2つの円を交互にクリックし、ミス回数と所要時間を記録する。

## アプリ仕様

### 画面構成

| 画面 | 内容 |
|------|------|
| TOPページ | 学籍番号入力、ターゲットサイズ選択、クリック回数選択 |
| 実験画面 | 500px間隔の2つの円を交互にクリック。最初のクリックで計測開始 |
| 結果画面 | 学籍番号・サイズ・ミス回数・所要時間を表示。スプレッドシート保存・CSV出力 |

### ターゲットサイズ

| ラベル | 直径 |
|--------|------|
| 最大 | 25px |
| 大 | 20px |
| 中 | 15px |
| 小 | 10px |

### クリック回数

10 / 20（デフォルト） / 30 / 50 から選択

### 記録データ

| 項目 | 説明 |
|------|------|
| 学籍番号 | TOPページで入力 |
| サイズ(px) | 選択したターゲット直径 |
| クリック回数 | 設定した目標回数 |
| ミス回数 | ターゲット外をクリックした回数 |
| 所要時間 | 最初のクリックから最後のクリックまでの時間（秒） |

### ファイル構成

```
fitts/
├── index.html   # 全画面（TOP・実験・結果）
├── style.css    # スタイル
├── app.js       # メインロジック
├── config.js    # GAS URL / スプレッドシートURL設定
└── gas-code.js  # Google Apps Script 用コード
```

## 開発者の設定手順

### 1. Google スプレッドシート作成

1. Google スプレッドシートを新規作成
2. 1行目にヘッダーを入力:

| A1 | B1 | C1 | D1 | E1 |
|----|----|----|----|----|
| 学籍番号 | サイズ(px) | クリック回数 | ミス回数 | 所要時間 |

### 2. Google Apps Script 設定

1. https://script.google.com にアクセス
2. 「新しいプロジェクト」を作成
3. `gas-code.js` の `doPost` 関数を貼り付け
4. コード内の `'スプレッドシートIDをここに入れる'` を実際のIDに差し替え
   - スプレッドシートURLが `https://docs.google.com/spreadsheets/d/XXXXX/edit` の場合、`XXXXX` がID
5. 保存（Ctrl+S）

### 3. GAS デプロイ

1. 「デプロイ」→「新しいデプロイ」
2. 歯車アイコン →「ウェブアプリ」を選択
3. 実行ユーザー: **自分** / アクセスできるユーザー: **全員**
4. 「デプロイ」→ アクセス承認（Google アカウント選択 → 詳細 → 安全でないページに移動 → 許可）
5. 表示されたウェブアプリURLをコピー

### 4. config.js の設定

```js
const CONFIG = {
  GAS_URL: 'デプロイで取得したウェブアプリURL',
  SHEET_URL: 'スプレッドシートの共有URL',
};
```
